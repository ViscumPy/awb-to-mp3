# awb-to-mp3

只是为了打astrodx有更好的体验才出现的东西

## 前置

```bash
pip install -r requirements.txt
```

## 使用方法
- **awb转mp3**: 将`awb`丢入`input`，双击`启动.bat`，选择模式
- **acb转mp3**: 将`acb`丢入`input`，双击`启动.bat`，选择模式

## 其他说明
- **python版本**: 3.9+
- **NET版本**: 8.0
- **acb**: 在`sinmai`中，`acb`存储着系统音效的音频，如：`Mai2Cue.acb`
- **awb**: 在`sinmai`中，`awb`存储着乐曲音乐的音频，如: `music000008.awb`
